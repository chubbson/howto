#### Basics

Headings: `*`, `**`, `***` - different levels
Lists bullets: `-`. `+`
Lists checkbox: `[ ]`, `[X]` for tasks
Checkboxes: `[ ]`. 
- `[X]` for done
- `[-]` for in progress
Links: `[[URL][Description]]`
Basing Commands:
* cycle states: `c-c c-t`
* update field/boxes: `c-c c-c`
* shedule task: `c-c c-s`
* set deadline: `c-c c-d`

#### configuration

chekout `~/.emacs` or `~/.emacs.d/init.el`
