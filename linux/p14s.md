# Lenovo P14s Gen 5 - Arch Linux Setup

Tags: #p14s #hardware #setup

## Hardware

| Component | Spec |
|-----------|------|
| Model | Lenovo ThinkPad P14s Gen 5 Intel |
| CPU | Intel Core Ultra 9 185H |
| GPU | Intel Arc 140V (integrated) + GTX 1070 (eGPU via Thunderbolt) |
| RAM | 96GB |
| Storage | 3.6TB NVMe |

## Disk Layout

```
nvme0n1p1  512MB   EFI (unencrypted)
                   └── GRUB + kernel + initramfs
nvme0n1p2  100GB   Free (future dual boot)
nvme0n1p3  ~3.3TB  LUKS encrypted (Argon2id)
                   └── LVM vg0
                       ├── swap    100GB
                       ├── root    400GB  btrfs
                       ├── home    2TB    btrfs
                       └── vm      923GB  xfs
nvme0n1p4  2GB     winRE (untouched)
```

See [[luks]] for encryption details.

## Boot Flow

```
UEFI → GRUB (silent)
→ kernel starts
→ PIN prompt → SoloKey touch OR Enter → passphrase
→ LUKS unlocks → Arch boots
```

See [[solo2]] for SoloKey setup.

## Filesystem

### btrfs subvolumes

```
vg0-root
├── @           → /
├── @snapshots  → /.snapshots
└── @var_log    → /var/log
vg0-home
└── @home       → /home
```

- zstd compression enabled
- SSD optimizations enabled

## GPU Notes

- Intel Arc 140V requires kernel parameter: `i915.force_probe=<device-id>`
- eGPU (GTX 1070) authorized via bolt (Thunderbolt)

See [[arch]] for kernel parameter setup.

## Dev Environment Architecture

```
Arch Linux (host)
├── Docker
│   └── SQL Server (multiple versions)
│       └── mounts /vm/sqldata (.mdf/.ldf files)
├── KVM/QEMU Windows VM
│   ├── Visual Studio (multiple versions)
│   ├── Teams + Outlook
│   └── SSMS → connects to SQL Server via 192.168.122.1
└── /vm partition (xfs, 923GB)
    ├── windows.qcow2
    └── sqldata/
```

See [[kvm]] and [[docker]].
