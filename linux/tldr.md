# tldr

Simplified, practical man pages with examples.

Tags: #shell #cli #docs

```bash
sudo pacman -S tldr
```

## Setup

Update cache before first use:
```bash
tldr -u
```

## Usage

```bash
tldr tar             # quick examples for tar
tldr -u              # update local cache
tldr -l              # list all available pages
tldr -s keyword      # search by keyword
tldr -p linux tar    # specify platform (linux, osx, windows, common)
```
