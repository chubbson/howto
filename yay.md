yay [[AUR]] Helper for Arch Linux

install: 
`sudo pacman -S yay`

install package using yay
`yay -S package`

remove package using yay
`yay -Rns packager`

update your [[Arch]] system with inlc packages
`yay -Syu`

Clean unneeded dependencies
`yay -Yc`

Print system statistics
`yay -Ps`
